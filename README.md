# RestaurantIQ

Plataforma SaaS premium de gestão para restaurantes, com IA integrada para apoiar decisões financeiras e operacionais.

## Sobre o projeto
O RestaurantIQ centraliza a gestão de um restaurante em um único painel: controle de CMV, fichas técnicas, estoque, fator de correção (FC), DRE e fornecedores — com uma assistente de IA (Oliveira) analisando os dados e sugerindo ações.

## Tecnologias
- **Frontend**: Next.js 14 (App Router), TailwindCSS
- **Backend/DB**: PostgreSQL via Supabase
- **IA**: OpenAI GPT-4o
- **Pagamentos**: Stripe

## Funcionalidades
- Gestão de fichas técnicas e cálculo de CMV
- Base de fator de correção com 80+ ingredientes (fontes acadêmicas: UFPR, NEPA-UNICAMP/TACO, Ornellas)
- Controle de estoque e fornecedores
- Dashboards financeiros (DRE)
- Assistente de IA para análise e recomendações
